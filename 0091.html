Subject: Прямоугольные треугольники с целыми координатами Body:

Точки P (x1, y1) и Q (x2, y2) находятся на целых координатах и соединены с началом координат O(0,0), образуя треугольник ΔOPQ.


Существует ровно четырнадцать треугольников с прямым углом, которые можно получить для целых координат в пределах от 0 до 2 включительно, т.е.
0 ≤ x1, y1, x2, y2 ≤ 2.


При данных 0 ≤ x1, y1, x2, y2 ≤ 50, сколько прямоугольных треугольников можно построить?
