Для каждого целого n1, семья функций fn,a,b задана как fn,a,b(x)ax+b mod n для целых a,b,x и 0an, 0bn, 0xn.

Пусть R(n) будет количеством втягиваний для n.

F(N)=R(n) для 2nN.
