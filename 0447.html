Subject: Втягивания C Body:

Для каждого целого n>1, семья функций fn,a,b задана как fn,a,b(x)≡ax+b mod n для целых a,b,x и 0x Назовем fn,a,b втягиванием, если fn,a,b(fn,a,b(x))≡fn,a,b(x) mod n для каждого 0≤x Пусть R(n) будет количеством втягиваний для n.

F(N)=∑R(n) для 2≤n≤N.
