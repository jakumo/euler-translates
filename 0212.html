Subject: Общий объем кубоидов. Body:

Выравненный по осям кубоид, определяемый параметрами { (x0,y0,z0), (dx,dy,dz) }, состоит из таких точек (X,Y,Z), для которых x0 X x0+dx, y0 Y y0+dy и z0 Z z0+dz. Объем кубоида есть произведение dx dy dz. Общий объем набора кубоидов равен объему объединенных кубоидов, и, в случае перекрытия кубоидов, он будет меньше суммы отдельных объемов.

Пусть C1,...,C50000 - набор 50000 выравненных кубоидов, параметры каждого кубоида Cn представлены ниже:

x0 = S6n-5 modulo 10000
y0 = S6n-4 modulo 10000
z0 = S6n-3 modulo 10000
dx = 1 + (S6n-2 modulo 399)
dy = 1 + (S6n-1 modulo 399)
dz = 1 + (S6n modulo 399)

где S1,...,S300000 получают с помощью "Генератора Фибоначчи с задержкой":

For 1 k 55, Sk = [100003 - 200003k + 300007k3]   (modulo 1000000)
For 56 k, Sk = [Sk-24 + Sk-55]   (modulo 1000000)

Таким образом, параметры C1: {(7,53,183),(94,369,56)}, параметры C2: {(2383,3563,5079),(42,212,344)}, и т.д.

Общий объем первых 100 кубоидов, C1,...,C100, равен 723581599.

Чему равен общий объем всех 50000 кубоидов, C1,...,C50000 ?
