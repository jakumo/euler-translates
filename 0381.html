Для простого числа p пусть S(p) = ( ∑ (p-k)!) mod(p) для 1 ≤ k ≤ 5.

Например, если p=7,
(7-1)! + (7-2)! + (7-3)! + (7-4)! + (7-5)! = 6! + 5! + 4! + 3! + 2! = 720+120+24+6+2 = 872.
Так как 872 mod(7) = 4, S(7) = 4.

Можно убедиться, что ∑ S(p) = 480 для 5 ≤ p < 100.

Найдите ∑ S(p) для 5 ≤ p < 108.
