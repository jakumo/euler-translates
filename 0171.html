Subject: Ищем числа, для которых сумма квадратов цифр являтся квадратом Body:

Для положительного целого числа n, пусть f(n) будет суммой квадратов его цифр (по основанию 10), т.е.

f(3) = 32 = 9,
f(25) = 22 + 52 = 4 + 25 = 29,
f(442) = 42 + 42 + 22 = 16 + 16 + 4 = 36

Найдите последние девять цифр суммы всех n, 0 < n < 1020, таких, что f(n) является квадратом целого числа.
